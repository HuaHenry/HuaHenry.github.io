句子系统（System of sentences）

句子系统：一组为你提供信息的句子 和线性方程组提供信息的方式非常像

  • 完备系统：句子和提供的信息量相同
  • 冗余系统：可以用更少的句子表示相同的信息
  • 矛盾系统：句子之间信息矛盾

其中完备系统是非奇异的（Non-singular），冗余和矛盾系统都是奇异的（Singular）。

对于不同冗余程度的衡量，可以使用秩（Rank）。

方程组（System of equations）

方程组就像上面的句子一样提供信息，也可以是奇异或非奇异的，取决于他们所包含的信息量。

和句子系统相似，在系统没有足够信息时会出现无穷多解（无法缩小到完全系统）和无解

线性方程组和非线性方程组的区别

线性代数（以下所有）是研究线性方程组的学科。

方程组作为直线和平面

二维线性方程（组）几何表示：线性方程是空间中的直线，线性方程组是平面上直线的排列。

矩阵行化简

矩阵行简化（也称高斯消元）：对矩阵的行施加完全相同的操作，使得更为简化后可以求解线性方程组

求解线性方程组的过程就是行简化：原矩阵 上对角矩阵 对角矩阵

  • 上对角矩阵：行阶梯式（Row echelon form）
  • 对角型矩阵：简化行阶梯式（Reduced row echelon form）

其中行阶梯式作为中间过程可以给求解提供许多有用信息。

行阶梯式的一般形式：主对角线先1后0（均可无），1后可以是任意数，0后必须是0

保持奇异性的行操作

行简化（操作）的重要性质：矩阵奇异性能够得以保持

行操作的类型：

  • 对换两行（行列式值为原来值相负）
  • 某行乘非零数（行列式值对应为原来的非零数倍）
  • 某行乘任意倍加到另一行（行列式值不变）

由于通过以上操作，对应矩阵的行列式都不会改变非零或零的性质，所以行操作能保持奇异性。

矩阵的秩（Rank）

秩：衡量和矩阵（及其对应的线性方程组）包含的信息量。

如数字图像处理中图像压缩的秩（原图相当于像素矩阵，通过SVD降低矩阵的秩以达到压缩目的）

对应线性方程组（和矩阵），秩表示的是包含有效不冗余信息的方程的个数：

解空间：右侧常数为0时方程组的解

满秩的概念：矩阵的秩等于行数。

当矩阵满秩，矩阵是非奇异矩阵，线性方程组有唯一解；反之为奇异矩阵，线性方程组无穷解或无解。

当且仅当行阶梯式主对角线只有 1 没有 0 时，矩阵是非奇异的。

一般行阶梯型

一般的行阶梯型遵循：

  1. 所有 0 行都在最后行
  2. 每行最左侧非零元为主元（pivot）
  3. 每行主元必须在上一行主元的右侧

结论：矩阵的秩 = 主元的数量

简化行阶梯型

化为行阶梯形后额外进一步简化为主对角线全为 1、其余地方全为 0 的矩阵 简化行阶梯形，对应于方程组解的系统。

转换过程即为行简化操作，用下一行的主元消去上一行的非主元。

高斯消元法（Gaussian Elimination）

增广矩阵（Augmented matrix）：将常数项增添在系数矩阵右侧的矩阵

对增广矩阵实现行简化操作，化为系数矩阵为行最简形（行简化阶梯型）的形式，对应增广矩阵的右侧部分就是线性方程组的解。

求解过程中若系数矩阵化出全零行 矩阵是奇异的，无求解必要。但是仍然可以判断是无穷解还是无解：

  • 若零行对应增广矩阵右侧元素也是0：冗余，无穷解
  • 若为非0：矛盾，无解
